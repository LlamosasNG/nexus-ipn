# Repository Map

## Important directories

/client/src/api
Frontend API communication layer

/client/src/views
Application screens

/client/src/components
Reusable UI components

/server/src/controllers
Business logic handlers

/server/src/routes
Express routes

/server/src/models
Database models

/server/src/middleware
Auth and validation middleware

## Ignore

Do not analyze:
- /dist
- /node_modules
- generated assets
